Introdução ao HTML na Prática

Caro(a) leitor(a), seja bem-vindo(a) a essa introdução de HTML! Para dar inicio, você sabe o que significa HTML?

O HTML significa Linguagem de Marcação de HiperTexto, é o bloco de construção mais básico da Web. Ele permite criar páginas como esta que você está lendo agora. Venha comigo para explorarmos juntos as principais ferramentas e técnicas para criar suas próprias páginas!

Ferramentas utilizadas

Para começar a criar páginas Web, você precisará de um editor de código. As opções mais populares são o Sublime Text e o Visual Studio Code. Esses editores oferecem uma interface simples e são fáceis de entender. Além disso, você pode instalar plugins que facilitam muito o trabalho de programação.

Dois plugins essenciais são:

  • Live Server: Esse plugin cria um LocalHost que atualiza automaticamente o navegador sempre que você salva seu projeto, evitando a necessidade de atualizar a página manualmente.
  • Emmet: Já vem pré-instalado. Com ele, você pode usar atalhos como html:5 para gerar todos os códigos básicos iniciais do HTML em segundos.

Inspetor de elemetos

O Inspetor de Elementos é uma ferramenta poderosa que permite visualizar e modificar temporariamente o código-fonte de qualquer página Web. É uma maneira fantástica de aprender e testar ideias sem alterar o código original.

Para acessar o inspetor, pressione F12 no seu teclado e depois Ctrl + Shift + C. Em seguida, passe o cursor sobre os elementos da página para ver o código correspondente na janela de inspeção.

Estrutura básica do HTML

Antes de começar a escrever código, é importante entender a estrutura básica do HTML. Todo documento HTML é composto por três partes principais:

  • <html></html>: A tag que envolve todo o conteúdo da página, indicando que estamos trabalhando com HTML. Essa tag abre no inicio e fecha no final do documento.
  • <head></head>: Contém informações sobre a página, como o título, que aparece na aba do navegador, e links para arquivos CSS.
  • <body></body>: Aqui vai todo o conteúdo visível da página, como textos, imagens, links, etc.

Quando você começa a criar um site, o primeiro arquivo que você deve criar é chamado de index.html. Mas por que exatamente o index.html?

O arquivo index.html é o padrão usado pelos servidores Web para identificar a página inicial de um site. Quando um usuário acessa um domínio, como www.exemplo.com, o servidor Web automaticamente procura por um arquivo chamado index.html na pasta raiz do site. Se ele encontrar esse arquivo, será o primeiro a ser carregado, servindo como a porta de entrada para o restante do conteúdo do site.

Isso torna o index.html uma peça fundamental na estrutura do seu site, pois ele funciona como uma "capa" ou "home" que organiza e guia os usuários para as outras páginas e seções. Portanto, sempre comece seus projetos criando um arquivo index.html para garantir que seu site funcione corretamente e seja acessível da maneira esperada.

Tags de Formatação

As tags de formatação ajudam a estilizar o conteúdo da sua página. Algumas das mais usadas são:

  • <strong></strong>: Deixa o texto em negrito.
  • <mark></mark>: Destaca o texto como se fosse um marca-texto.
  • <font></strong>: Usado para definor a face, cor ou tamanho da fonte no texto (está desatualizado, mas ainda é aceito em muitos navegadores).

Existem também tags que não precisam ser fechadas por não terem conteúdo interno, ou seja, não contém textos ou outros elementos que precisem ser delimitados. Elas são usadas para inserir algo diretamente na página, como um campo de entrada ou imagem:

  • <abbr>: É uma abreviação. Quando o cursor repousa sobre a palavra, o usuário pode visualizar o significado ou explicação no balão que aparece.
  • <input>: Cria uma caixa de texto.


  • <img>: Insere uma imagem na página. Essa tag é referenciada por um atributo (src).

Atributos das Tags

Atributos são usados para fornecer informações adicionais às tags. Alguns dos atributos mais comuns são:

  • id: Identifica de forma única um elemento na página, útil para estilização e navegação.
  • style: Aplica estilos diretamente a uma tag. Exemplo: <p style="color: blue;">Texto Azul</p>.
  • class: Agrupa elementos para aplicar estilos ou manipulações em conjunto. Exemplo: <p class="destaque">Destaque</p>.
  • input: Define o tipo de dado que pode ser inserido em uma caixa de texto. Exemplo: <input type="number"> para aceitar apenas números ou <input type="color"> para abrir uma caixa seletora de cores.
  • width: Define a largura de elementos como imagens. Pode ser definido um tamanho fixo, usando 150px, ou percentualmente proporciional ao tamanho da página, usando 50%. Exemplo: <img src="imagem.jpg" width="50%">.

Textos

O HTML oferece várias tags para formatação de textos. Aqui estão algumas das mais utilizadas:

  • <i></i>: Deixa o texto em itálico.
  • <u></u>: Sublinha o texto.
  • <h1> a <h6>: Definem títulos, com <h1> sendo o maior e <h6> o menor.
  • <p>: Cria um parágrafo de texto.
  • <blockquote>: Usado para citações. Combine com <i> para dar ênfase.
  • <sup>: Exibe texto sobrescrito, como em 102.
  • <sub>: Exibe texto subscrito, como em H2O.
  • <del>: Exibe texto tachado. Muito usado para destacar exclusão de palavras ou trechos.

Listas Ordenadas e Não Ordenadas

As listas são uma forma eficiente de organizar conteúdo. No HTML, podemos criar tanto listas ordenadas quanto não ordenadas, dependendo da necessidade.

Lista Ordenada

Uma lista ordenada é útil quando a sequência dos itens importa, como em um passo a passo. Para criar uma lista ordenada, usamos as tags <ol> e <li>:

  1. Primeiro item
  2. Segundo item
  3. Terceiro item

Lista Não Ordenada

Se a ordem não é importante, você pode optar por uma lista não ordenada, que usa marcadores em vez de números. Aqui, utilizamos as tags <ul> e <li>:

  • Item A
  • Item B
  • Item C

Tanto em listas ordenadas quanto em não ordenadas, a tag <li> é fundamental para definir cada item da lista.

Links

Os links são a alma da Web, permitindo a navegação entre páginas e dentro da mesma página. A tag <a> (âncora) junto com o atributo href (referência de hiperlink) é usada para criar essas conexões.

Link Externo

Para criar um link para um site externo, utilizamos o atributo href com a URL completa do site: 

<a href="https://www.dio.me/" target="_blank">DIO</a>

Uma dica é acrescentar o atributo 'target="_blank"' para que, ao clicar no link, o usuário abra outra aba, não encerrando a navegação do seu site*++ .

Link Interno

Se quisermos navegar dentro do próprio site, basta usar um caminho relativo. Por exemplo, para linkar à página "link-interno.html", fazemos assim: 

<a href="link-interno.html">Link Interno</a>

Links na Mesma Página

Para navegar dentro da mesma página, utilizamos IDs. Primeiro, criamos o link: 

<a href="#links">Links</a>

E depois, definimos o ID no destino: 

<h2 id="links">Links</h2>

Como dica para facilitar a navegação, podemos adicionar um link para voltar ao topo da página: 

<small><a href="#topo">Voltar ao topo</a></small>
